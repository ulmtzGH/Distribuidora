# Distribuidora

Proyecto desarrollado en ASP.NET Core 8 utilizando arquitectura en capas.

## Tecnologías

- ASP.NET Core MVC
- ASP.NET Core Web API
- SQL Server
- ADO.NET
- Stored Procedures
- Bootstrap 5

## Arquitectura

```
MVC
 ↓
API
 ↓
Business
 ↓
Data
 ↓
SQL Server
```

## Base de datos

SQL Server

Toda la comunicación con la BD se realiza mediante Stored Procedures.
